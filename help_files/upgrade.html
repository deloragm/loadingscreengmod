UPGRADE

I've tried to make upgrading as easy as possible, below are guides on how to upgrade your specific version of FlatLoad

UPGRADE FROM V1.4 TO V1.5

Upgrading your version of FlatLoad 1.4 to 1.5 is very easy but if you don't want the hassle with upgrading then you can just use the already upgrade versions in this download and just reconfigure it with the same settings you used previously. To upgrade your old version all you will need to do is edit index.php and style.css

1. EDIT CODE IN STYLE.CSS
1.5 fixes a weird rendering issue with the border on the avatar so you will need to open up style.css and look for the code below.

#avatarimg {border: 2px solid #FFFFFF; border-radius: 50%; width: 170px; height: 170px; 
float: left; margin: 15px 0 0 30px;}

and replace it with the code below.

#avatarimg {border-radius: 85px; -moz-border-radius: 85px; -webkit-border-radius: 85px; 
overflow: hidden; width: 170px; height: 170px; float: left; margin: 15px 0 0 30px; 
border: 2px solid #FFFFFF;}

#avatarimg img {width: 170px; height: 170px;}

2. EDIT INDEX.PHP You now need to open up index.php and look for the code below.

echo "<img id=\"avatarimg\" src=\"" . $table["avatarfull"] . "\" />";

and replace it with the code below:

echo "<div id=\"avatarimg\">";
echo "<img src=\"" . $table["avatarfull"] . "\" />";
echo "</div>";

3. YOUR DONE
That's it!, we're all done, if something has gone wrong then it's probably easier to do a new installation.

UPGRADE FROM V1.3 TO V1.4

Upgrading your version of FlatLoad 1.3 to 1.4 is very easy but if you don't want to hassle with upgrading then you can download 1.4 and just reconfigure it with the same settings you used previously.

1. ADD NEW STYLESHEETS
In V1.4 there are now two stylesheets in your core loading screen folder, these are style.css and colour.css, these two files define how the loading page should look so you need to copy these two new and updated files into your exisiting version.

2. ADD CODE TO INDEX.PHP In V1.4 I have split some of the styling into a new CSS file called colour.css so as to make it easier for you to change the colour of the header bar.

We now need to link up this new stylesheet so we need to load up index.php and look for the little snippet of code below 

<link href="style.css" rel="stylesheet" type="text/css" /> <!-- Links to the Stylesheet -->

and all we need to do is add a little bit of code underneath it, below shows what it should look like.

<link href="style.css" rel="stylesheet" type="text/css" /> <!-- Links to the Stylesheet -->
<link href="colour.css" rel="stylesheet" type="text/css" />

3. YOUR DONE
That's it!, that's how easy it is to upgrade your existing installation, you can now take advantage of the multiple coloured backgrounds feature added in 1.4

To learn how to take advantage of this new feature just open up the instructions.
